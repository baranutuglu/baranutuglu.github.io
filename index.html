CSS Temel Ödevi

Verilen görsele benzer şekilde basit bir web sayfası oluştururken CSS'in temel özelliklerini uygulama.

İçerik

Bu alanda örnek metin bulunmaktadır. CSS öğelerini düzenleme işlemi yapılmıştır. Font boyutu, renk ve satır yüksekliği gibi özellikler uygulanmıştır. Web sayfasının temel tasarımını anlamak için bu örneği inceleyebilirsiniz.

Önceki CSS ile kutuları / alanları yan yana getirmeyi öğrendik. Şimdi CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapabiliriz. Bölümleri grup ve hiyerarşik seçiciler kullanarak stillendirebiliriz.

Yan Menü / Bilgi

Bu alan yan menü ve bilgi bölümü için ayrılmıştır. CSS ile geniş alanı (width), kenar farkları (margin/padding) ve kenarlık (border) özellikleri düzenlenmiştir. Metinlerin hizalaması da text-align özelliği ile yapılmıştır.

Öğrendiklerimizi başka seçiciler ve CSS özellikleriyle genişletebiliriz. Gradyan renkler, gölge efektleri ve animasyonlar eklenebilir. CSS'in gücünü kullanarak daha etkileyici tasarımlar oluşturabiliriz.
